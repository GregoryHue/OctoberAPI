# OctoberAPI

A small API with Node/Express

## Project setup

Install the libraries with:

```
cd api/ && npm install
```

## Usage

Run the API with:

```
cd api/ && npm run start
```

Then go to: [localhost:3000/company/EXPERDECO/303830244](localhost:3000/company/EXPERDECO/303830244)

## Versions

* Node 16.16.0
* Npm 8.11.0
* Ubuntu 20.04.5
